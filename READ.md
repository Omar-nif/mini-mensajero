# mini-mensajero

## __Objetivo:__ una guia para hacer un proyecto web simple en el cual podamos apreciar como es el flujo que siguen los datos por toda la aplicacion.


### Pasos a seguir:

1. __Crear el proyeto:__ Primero pondremos en la terminal el comando __npm init -y__ con el cual crearemos un archivo JSON que servira de registro para las dependencias usadas en el proyecto. 

2. __Descargar dependencias:__ Descargamos dependencias como express, corss, etc....

3. __Activamos modulos ES:__ En el package.json agregamos `"type": "module"` para poder usar cosas como import y export.
